# ANALYZING, VISUALIZING AND PREDICTION OF APARTMENT PRICE (BUENOS AIRES REAL ESTATE)

In this project, I wants to create a model that can predict the price of apartments in the city of Buenos Aires

# PROJECT OVERVIEW

This project analyzes and predict apartment price in buenos aires real estate apartments using machine learning models. This price prediction model would be build based on three key fundamental dependent variable -size, location, neighbourhood and everything else.  The goal is to determine factors and variables with great impact on apartment prices in buenos aires. This will support:

+ Valuation of real estate property

+ Investment decisions for investors

+ Pricing strategies for both buyers and sellers

+ General analysis of the real estate industry

# BUSINESS PROBLEM
The pricing of real estate property is affected by multiple factors including location, property sizer, neighborhood characteristics, and amenities. Incorrect pricing of real estate properties can  lead to poor decision making from both the buyer and the seller, loss of potential buyers, underpricing of valuable properties

The core question that this project answers is that: 
    Can we accurately predict apartment prices using available features of the property?

Targeted audience is real estate buyers and house agents
